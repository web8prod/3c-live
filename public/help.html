3C Live の使い方

セッション会場で参加者がスマホから付箋を投稿、主催者がその場で集計・AI 分析までできるオンライン 3C 分析ツールです。

📍 全体像（1分で分かる）

①

主催者がセッション作成

管理画面でセッション名・日時を入力 → 6桁コードが発行されます

②

実施画面で QR 共有

実施画面を大画面に投影 → 参加者がスマホで QR 読み取り

③

参加者が付箋投稿

顧客・競合・自社の各テーマに思ったことを書く → リアルタイムで全員の画面に反映

④

分析画面で AI 提案

キーワード集計・類似クラスター → Claude AI で未来バリュープロポジションを3案生成

🎬 4つの画面

🔒 管理画面 /admin

誰が見る？ Web8 社員のみ（パスワード保護）

パスワード: Web82525

何ができる？

  • 新しいセッションを作る（タイトル・日時・主催者名・メモ）
  • 過去・予定のセッション一覧を確認
  • 各セッションの「実施」「分析」画面へジャンプ
  • 不要なセッションを一覧から外す

🎤 実施画面 /host?code=XXXXXX

誰が見る？ 主催者がプロジェクターやモニターに投影

何が映る？

  • 大きな QR コード（参加者がスマホで読む）
  • 6桁の参加コード（手入力でも入れる人向け）
  • 参加者一覧（誰が入室済みか）
  • 顧客・競合・自社の 付箋枚数カウント
  • 投稿された付箋の ライブフィード（新着が浮き上がる）

✋ 受講者画面 /?code=XXXXXX または /room?code=XXXXXX

誰が見る？ セッション参加者（誰でもアクセス可）

使い方:

  • QR 読み取り or 6桁コード入力 → 名前入力
  • 顧客（欲求/不満/不安/不便）・競合（印象/特徴）・自社（挑戦したいこと/できそうなこと/やりたいこと/経験/資産/資源/リソース/その他）の各テーマに付箋投稿
  • 「+ 自分でテーマを作って追加」で独自テーマも作れる（例: コスト感）
  • 付箋表示モード「付箋⇔リスト」切替可能

📊 分析画面 /analyze?code=XXXXXX

誰が見る？ 主催者（セッション終了後、または途中の中間集計）

何ができる？

  • 顧客／競合／自社それぞれの 頻出キーワード Top 15（バーグラフ）
  • 意見が似ている付箋の 自動クラスター
  • セッション全体を JSON / CSV 書き出し（記録保存用）
  • 🟢 無料: 「📋 プロンプトをコピー」→ Claude.ai に貼って AI 分析（推奨）
  • 有料: Anthropic API キー入力でアプリ内ワンクリック生成

📋 主催者：当日の流れ

  1. 前日まで: 管理画面 でセッションを作成（タイトル＝何のイベントか、日時、主催者名）
  2. 当日 開始10分前: 会場の PC で実施画面を開いて投影。Wi-Fi が安定していることを確認
  3. セッション開始: 参加者に「画面の QR を読み取ってください」と案内
  4. 進行中: 各カテゴリで参加者に書いてもらう。実施画面のライブフィードで盛り上がりが見える
  5. 振り返り: 分析画面を別タブで開いてキーワード集計とクラスターを見せる
  6. AI 提案: 「プロンプトをコピー」→ Claude.ai に貼って3案出させる → 参加者と議論
  7. 記録: JSON / CSV 書き出しでデータをアーカイブ

❓ よくある質問（FAQ）

Q. 管理画面のパスワードは？

A. Web82525 です。Web8 社員で共有してください。 パスワードはブラウザのセッションに保存されるので、タブを開き直したら再入力が必要です。

Q. パスワードを変更したい

A. 開発担当に依頼してください。public/js/auth-guard.js の EXPECTED_HASH 定数を新パスワードのハッシュに置き換えて GitHub に push すると Vercel が自動再デプロイします。

Q. AI 提案を使うのに API キーが必要？

A. 不要です。分析画面の緑のボタン「📋 プロンプトをコピー」を押すと、整形済みのプロンプトがクリップボードに入るので、それを claude.ai や ChatGPT の無料プランに貼り付けるだけで分析できます。

有料の API キーは「ワンクリック・サイト内完結」が欲しい場合のみ。

Q. 参加者の人数に上限はある？

A. 技術的には数百人まで OK ですが、実用的には30〜50人くらいまでが運用しやすいです（人数が多いとライブフィードが流れる速度が速く、議論しにくくなる）。

Q. 参加者が間違えて投稿した付箋を消すには？

A. 投稿者本人が自分の画面で付箋にカーソルを合わせると「✎ 編集」「✕ 削除」のボタンが出ます。他人の付箋は消せません。

Q. セッション終了後もデータは残る？

A. はい、永続的に Supabase に保存されます。後日、分析画面の URL（/analyze?code=XXXXXX）を開けば同じデータを参照できます。「削除」ボタンは一覧から外すだけで、データ本体は消えません。

Q. オフラインでも使える？

A. いいえ、参加者全員がインターネット接続している必要があります（Supabase クラウド DB を介してリアルタイム同期するため）。会場のWi-Fi が弱い場合は、各自スマホの 4G/5G 接続を案内してください。

Q. 「セミナー」じゃない用途でも使える？

A. もちろん。**社内ブレスト、合宿、ワークショップ、戦略会議**など、3C 分析が必要な場面ならどこでも使えます。セッション名は自由入力なので「2026/5/14 中小企業オーナー向け勉強会」とか「Q2 戦略会議」とか、開催の中身が分かる名前にしてください。

Q. 受講者画面の URL を直接スマホで開いてもらえる？

A. はい。https://3c-live-app.vercel.app/?code=XXXXXX の XXXXXX 部分を 6桁コードに置き換えて共有すれば、QR を経由せずに直接アクセス可能。Slack や Chatwork で URL を送ってください。

Q. 無料で使えるって本当？追加料金は？

A. はい、完全無料です。

  • Vercel (ホスティング): 無料枠で十分
  • Supabase (DB): 無料枠で 500MB DB / 2GB 通信／月
  • Claude.ai (AI 分析): 無料プランで OK
※ Anthropic API キー方式を使う場合のみ、1セッション数十円かかります（任意・推奨は無料の Claude.ai 経由）。

Q. 独自ドメイン（例: 3c.web8.jp）にしたい

A. Vercel ダッシュボード → Project Settings → Domains → Add domain で DNS の CNAME 設定をすれば数分で切り替わります。開発担当に依頼してください。

🛠 困ったときは

  • 付箋がリアルタイムで反映されない
    → Wi-Fi 接続を確認。ブラウザのリロード（⌘+R / Ctrl+R）で復旧することが多いです。
  • 管理画面でパスワードが通らない
    → 大文字小文字を確認。「Web82525」の頭文字は大文字 W、小文字 ebaid と紛らわしいです。
  • QR コードが読めない
    → 投影画面の解像度を上げる、または 6桁の参加コードを口頭で伝える。
  • 受講者画面で「この部屋は存在しません」と出る
    → 6桁コードのスペルミス・大文字小文字を確認。アルファベット O とゼロ 0、I と 1 が混ざりやすいです。
  • その他
    → 画面右下の 💬 ヘルプボタン から質問してください。または開発担当に直接連絡。

🔧 技術メモ（開発担当向け）

  • ホスティング: Vercel（GitHub web8prod/3c-live 連携、main へ push で自動再デプロイ）
  • データ: Supabase Postgres + Realtime（プロジェクト名 3c-live, リージョン Tokyo）
  • テーブル: rooms / notes / participants
  • セキュリティ: 管理画面はクライアント側パスワード、書き込みは Row Level Security + アプリ層 hostToken
  • AI 分析: Anthropic Claude（API キー or Claude.ai 経由）
